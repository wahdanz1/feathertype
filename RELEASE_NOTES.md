# FeatherType v1.0.0

A feather-light Markdown editor for desktop and web.

## Highlights

- **Live split-pane preview** — write Markdown on the left, see it rendered on the right
- **DOCX import** — open Word documents and convert them to Markdown instantly
- **Multi-tab editing** — work on multiple documents side by side
- **Dark and light themes** — switch with one click
- **Offline-first** — your documents stay on your device, never uploaded anywhere
- **Native file system** — open, save, and organize files through your OS (desktop)
- **Browser mode** — full editor available at [feathertype.vercel.app](https://feathertype.vercel.app/editor) with no install
- **Keyboard shortcuts** — formatting, saving, and navigation all from the keyboard
- **Unsaved changes tracking** — never lose work by accident
